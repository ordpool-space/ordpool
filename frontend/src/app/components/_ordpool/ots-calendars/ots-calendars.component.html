OpenTimestamps Calendars

Public calendar servers aggregate user file hashes into a Merkle tree once per Bitcoin block, then commit the root in an OP_RETURN. Every commit ordpool has indexed is shown below. Calendars are observability conveniences, not trust anchors -- once a `.ots` proof is upgraded, the calendar can disappear and the proof still verifies forever against any Bitcoin node. Learn more.
