OpenTimestamps Calendars

Bitcoin doesn't just store money. It stores time. OpenTimestamps lets you prove a file existed at a specific block by hashing it, batching that hash with thousands of others into a Merkle tree, and committing the root in a single OP_RETURN. That root is now part of Bitcoin's permanent ledger. Anyone with the file, the .ots proof, and Bitcoin's block headers can verify "this existed by block N", forever, with no third party in the loop. Photos, contracts, source code, leaked documents… anything with a hash gets a permanent, math-backed receipt. Ordpool tags every OTS transaction wherever it appears in the explorer, so you can see Bitcoin's notary at work in real time, and you can verify your own .ots receipt right here, against our independent open-source implementation.

Of course, the gold standard is to put the file itself on-chain as an inscription. We're 100% committed to inscriptions, that's what ordpool is for. But sometimes revealing the file isn't what you need: OTS commits only the hash, so the contents stay private and only you can ever prove what was timestamped. For everything you don't want to inscribe, OpenTimestamps is the next best thing: a tiny, permanent timestamp anchored in Bitcoin.
